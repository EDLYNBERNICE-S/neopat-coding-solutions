# NeoPAT / Neo IAM Coding Practice

This repository contains programming problems I solved on the **NeoPAT / Neo IAM platform** as part of my coding practice.

## Topics Covered

* Binary Search Tree
* Sorting Algorithms
* Problem Solving
* Basic Data Structures

## Problems

### 1. Delete Nodes Greater Than Average (BST)

Build a Binary Search Tree, compute the average of all nodes, and remove nodes greater than the average.

Concepts:

* BST insertion
* Tree traversal
* Conditional deletion

### 2. Temperature Sorting using Bubble Sort

Sort temperature readings (including negative values) in ascending order using Bubble Sort.

Concepts:

* Bubble Sort
* Array manipulation
